# FlashPipe 📱

FlashPipe is a macOS-native application built with **SwiftUI**, designed to simplify Android flashing tasks such as ADB detection, ROM sideloading, rooting, fastboot operations, and tool installation.

## ✨ Features
FlashPipe provides a streamlined interface for:
- **ADB Device Detection:** Instant recognition of connected devices. 🔍
- **ROM Sideloading:** Flash your favorite ROMs via ADB sideload with ease. ⚡
- **Fastboot Operations:** Manage bootloader commands directly from macOS. 🛠️
- **System Path Integration:** One-click tool installation for ADB/Fastboot. 🏗️
- **SwiftUI Interface:** A modern, native macOS experience. 🍏

## 📥 Getting Started

### Download & Releases
Click the link below to find the latest builds, source zips, and release notes:
👉 **[FlashPipe Releases](https://gitlab.com/ancientkira-group/flashpipe/-/releases/)** 📥

### Installation
1. Download the `FlashPipe_0.5_Beta.dmg` from the releases page.
2. Open the DMG and drag **FlashPipe** to your `/Applications` folder.
3. **Security Note:** Since this is an independent beta, you may need to right-click the app and select **Open**, or allow it in **System Settings > Privacy & Security** if macOS blocks the initial launch. 🛡️
