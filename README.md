# FlashPipe

FlashPipe is a macOS application designed to simplify Android flashing tasks such as ADB detection, ROM sideloading, rooting, fastboot operations, and tool installation.

---

## Project Status

**FlashPipe is currently under active development.**
A public release and full documentation will be added soon.

This repository may contain incomplete features, placeholders, or work in progress.

---

## About

FlashPipe provides:

- ADB device detection
- ROM flashing via ADB sideload
- Fastboot features
- Tool installation to system paths
- A macOS-native UI built with SwiftUI
