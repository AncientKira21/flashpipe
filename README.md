FlashPipe is a macOS-native application built with **SwiftUI**, designed to simplify Android flashing tasks such as ADB detection, ROM sideloading, rooting, fastboot operations, and tool installation.

## 🚀 Project Status: **v0.6-delta Released!**

**FlashPipe is currently near stable release (delta).** 🎊 

> [!NOTE]
> This repository is under active development. You may encounter incomplete features or placeholders as we work toward a stable 1.0 release.

---

## ✨ Features
FlashPipe provides a streamlined interface for:
- **ADB Device Detection:** Instant recognition of connected devices. 🔍
- **ROM Sideloading:** Flash your favorite ROMs via ADB sideload with ease. ⚡
- **Fastboot Operations:** Manage bootloader commands directly from macOS. 🛠️
- **System Path Integration:** One-click tool installation for ADB/Fastboot. 🏗️
- **SwiftUI Interface:** A modern, native macOS experience. 🍏
- **Pixel Bootloader Control (New):** Unlock or relock the bootloader on supported Pixel devices. 🔑📱
- ⚠️ **Warning:** Do **NOT** attempt on **Pixel 2 XL or older** devices.
  - Pixel 2 XL may appear *partially unlocked* and is **not supported**
  - **Minimum supported device:** **Pixel 3 or newer only**

## 📥 Getting Started

### Download & Releases
Click the link below to find the latest builds, source zips, and release notes:
👉 **[FlashPipe Releases](https://github.com/ancientkira/flashpipe/releases/)** 📥
👉 **[FlashPipe GitLab Mirror](https://gitlab.com/ancientkira/flashpipe/-/releases/)** 📥

### Installation
1. Download the `FlashPipe_0.6_RC.dmg` from the releases page.
2. Open the DMG and drag **FlashPipe** to your `/Applications` folder.
3. **Security Note:** Since I don't have a paid developer account, you may need to right-click the app and select **Open**, or allow it in **System Settings > Privacy & Security** if macOS blocks the initial launch. 🛡️

### Compatibility
The app requires macOS 14 or later. 💻⬆️
